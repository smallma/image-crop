# PMKT Image Studio

**在瀏覽器裡完成裁切、調整尺寸與格式轉換的圖片工具 — 圖片永遠不離開你的電腦。**

行銷團隊每天都在處理一件事：把同一張圖，調成 Instagram 要的正方形、Facebook 要的橫幅、YouTube 要的縮圖。PMKT Image Studio 把這件雜事變成幾秒鐘的點擊，而且**所有處理都在本機瀏覽器記憶體完成，不上傳任何伺服器**。

### 🔗 立即試用

**[image-crop-ten.vercel.app](https://image-crop-ten.vercel.app/)** — 免安裝、免註冊、打開就能用。

## 為什麼選它

| | |
|---|---|
| 🔒 **隱私優先** | 圖片只在你的瀏覽器處理，不上傳、不留存、不外洩。適合處理未公開的行銷素材。 |
| ✂️ **非破壞式自由裁切** | 拖曳框線自由裁切，隨時能拉回先前裁掉的內容，原圖永遠保留。 |
| 🖼️ **多圖同時工作** | 一次匯入多張圖片，每張獨立保存裁切、尺寸、旋轉與匯出設定，自由切換不丟失。 |
| 📐 **一鍵社群尺寸** | 內建 Instagram、Facebook、LinkedIn、YouTube 常用尺寸，選了就自動填滿。 |
| 🔄 **完整變形工具** | 旋轉、水平／垂直翻轉、縮放、拖曳定位，預覽與輸出位置完全一致。 |
| 💾 **多格式匯出** | JPG、PNG、WebP 任選，可調品質，檔名自動帶尺寸。 |

## 內建社群尺寸

| 平台 | 比例 | 尺寸 |
|---|---|---|
| Instagram 貼文 | 1:1 | 1080 × 1080 |
| Instagram 限時動態 | 9:16 | 1080 × 1920 |
| Facebook 貼文 | 1.91:1 | 1200 × 630 |
| LinkedIn 橫幅 | 4:1 | 1584 × 396 |
| YouTube 縮圖 | 16:9 | 1280 × 720 |

也支援自訂寬高（1–10000px），可鎖定長寬比例同步縮放。

---

## 使用流程

1. **拖曳或選擇圖片** — 支援 JPG、PNG、WebP，可一次多張。
2. **裁切與變形** — 自由裁切、旋轉、翻轉、縮放、拖曳定位。
3. **套用尺寸** — 點選社群預設，或手動輸入寬高。
4. **匯出下載** — 選擇格式與品質，一鍵下載。

全程在本機完成，沒有等待上傳、沒有檔案大小限制焦慮。

## 技術堆疊

純前端、零後端、靜態部署：

- **React 19** + **TypeScript**
- **Vite** 建置
- 原生 **Canvas API**（裁切與匯出運算）
- **Lucide** 圖示
- 原生 CSS（無 UI 框架）
- **Vercel** 部署

無障礙設計：所有文字最小 18px、對比至少 4.5:1（主要文字達 18.81:1）、鍵盤 focus 清晰、手機版不溢位。
